## Solve the Problem 

To solve the problem, we will need to collect the following.

*  Isolate the cause : Browser version, release, etc.  
*  Reproducible environment . 
*  Versions of Browser, server etc.
*  Screenshots of the issue.
*  Fiddler Logs.
*  Browser error logs.
